# mimic-mjx-web-demo

https://mimic-mjx-web-demo.pages.dev/


Real-time web demo for the [MIMIC-MJX](https://mimic-mjx.talmolab.org/) project, using [mujoco-js](https://www.npmjs.com/package/mujoco-js) and [onnxruntime-web](https://www.npmjs.com/package/onnxruntime-web). Written with help from Claude Code.
